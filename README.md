## Support runtime

* .net framework 3.5;
* .net framework 4.0;
* .net framework 4.5;
* .net standard 2.0;
* .net core app 2.0;

## Symbol.dll
* 通用特性扩展：ConstAttribute；
* 常用静态扩展；
* 各个支持时兼容性处理；

## Symbol.Data.dll
* 轻量级ORM，实体与查询无需一致；
* 支持T-SQL和NoSQL语法混用；
* 支持数据库架构版本检测；
* 数据库：MSSQL 2005/2008/2012/2014/2016+；
* 数据库：PostgreSQL，程序目录需要包含Npgsql.dll；
* 数据库：Mysql，程序目录需要包含Mysql.Data.dll；
* 数据库：SQLite，程序目录需要包含System.Data.SQLite.dll，可以分别存放x64/x86，自动识别；
